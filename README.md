YukkuriMovieMaker4（YMM4）上で動作する、入力映像の輝度から紙の階層を自動生成し、切り絵・紙工作風の立体感へ変換する映像エフェクトプラグインです。
単一の入力フレームだけで紙の段差、影、ハイライト、紙目を合成します。
暗い部分ほど手前の紙として扱い、階層境界に影を落とすことで、平面素材を重ね切りしたような見た目を作ります。
数値パラメータはアニメーションに対応しています。

## 概要

本プラグインは YMM4 の「映像エフェクト」として動作し、エフェクトの種類一覧では「ペーパーカットアウト」として表示されます。

Direct2D カスタムピクセルシェーダーがフレームごとに入力映像の輝度を求め、指定した階層数に量子化します。隣接画素との階層差から切り口を検出し、光の角度と奥行きに応じて影とハイライトを合成します。紙目は座標に基づく決定論的ノイズで生成するため、静止映像ではフレームごとにちらつきません。

このエフェクトは AviUtl 向けの EXO 出力（`.exo`）には対応していません。

## インストール方法

1. [Releases](https://github.com/routersys/YMM4-PaperCutout/releases/latest) ページから最新のプラグインファイル（`.ymme`）をダウンロードしてください。
2. YMM4 が起動していないことを確認し、ダウンロードしたファイルを実行してインストールします。
3. YMM4 を起動し、タイムライン上のアイテムに映像エフェクトを追加します。
4. 映像エフェクトの種類として **「ペーパーカットアウト」** を選択してください。

## 主な機能

### 1. 輝度からの階層化

入力映像の輝度をもとに、3 から 8 段の紙レイヤーへ自動分割します。明るい部分ほど奥、暗い部分ほど手前の紙として扱います。手動の点指定や参照画像は不要です。

### 2. 紙の段差と影

階層境界を切り口として扱い、奥行き、影、段差、光の角度から立体感を作ります。奥行きを上げると影が伸び、段差を上げると切り口の明暗差が強くなります。

### 3. 紙目

座標ベースのノイズで紙の繊維状のざらつきを加えます。紙目を上げると均一な塗りではなく、紙素材の粗さが出ます。

### 4. 元色と紙色の合成

元色で入力映像の色を残す割合を調整できます。0% では紙色寄り、100% では元映像の色を強く残します。紙色は基準となる色を指定します。

### 5. シンプルな調整項目

階層数、奥行き、影、段差、紙目、光の角度、元色、紙色、適用量だけで調整できます。複雑なマスクや素材の用意は不要です。

## パラメータ一覧

| パラメータ名 | 型 | デフォルト | スライダー表示範囲 | アニメーション | 説明 |
|---|---|---|---|---|---|
| 階層数 | 整数 | 5 | 3 〜 8 | ✗ | 輝度から自動生成する紙の階層数です。 |
| 奥行き | 数値 | 6px | 0 〜 24px | ✔ | 階層境界に落とす影の距離です。 |
| 影 | 数値 | 45% | 0 〜 100% | ✔ | 紙の段差で発生する影の強さです。 |
| 段差 | 数値 | 55% | 0 〜 100% | ✔ | 切り口の明暗差と立体感です。 |
| 紙目 | 数値 | 18% | 0 〜 100% | ✔ | 紙の繊維状のざらつきです。 |
| 光の角度 | 数値 | 135° | -180 〜 180° | ✔ | 影とハイライトの向きです。 |
| 元色 | 数値 | 70% | 0 〜 100% | ✔ | 元画像の色を残す割合です。 |
| 紙色 | 色 | `#FFF6EEDA` | — | ✗ | 紙として混ぜ込む基準色です。 |
| 適用量 | 数値 | 100% | 0 〜 100% | ✔ | 元画像とエフェクト結果の合成量です。 |

## 制限事項

- AviUtl 向けの EXO 出力（`.exo`）には対応していません。
- 透明部分（アルファが 0）には効果が適用されません。

---

## 注意事項

- **AviUtl 非対応**: 本プラグインは AviUtl 向けの EXO 出力（`.exo`）に対応していません。
- **透明部分の扱い**: ソース画像の透明部分（アルファが 0）はそのまま出力され、影や紙目は透明部分へ出ません。
- **紙目の再現性**: 紙目は座標から生成するため、同じ入力の同じ座標では常に同じ結果になります。
- **本プラグインを使用する前に、YMM4 プロジェクトファイルのバックアップを作成することを推奨します。**
